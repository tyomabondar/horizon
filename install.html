how to install horizon

1. download image

download the appropriate iso or image for your device (desktop, laptop, or phone).

2. prepare device

for desktops/laptops: create a new virtualbox vm or flash to a usb drive.
for phones: use the supported installer or flashable image (instructions included)(if you still want ios or android use UTM (ios) or limbo (android).

3. boot and install

boot your device and follow the installation prompts.

4. enjoy

once installation is complete, enjoy horizon on any device!
